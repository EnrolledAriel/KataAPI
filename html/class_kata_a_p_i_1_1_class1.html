static bool KataAPI.Class1.IsThueMorse ( int[]  seq)
staticprivate

"IsThueMorse" method decides whether a binary code is a Thue-Morse or not.
Thue-Morse is a infinite sequence and is obtained by having as a first value 0 and than repeatedly concatenating the current sequence with it's complement.

For more info about Thue-Morse, check-out: https://en.wikipedia.org/wiki/Thue%E2%80%93Morse_sequence.
The link to this Kata: https://www.codewars.com/kata/589a9792ea93aae1bf00001c.

INPUT:
Integer array "seq".
An non-empty array.

Constraints:
1 <= seq.length <= 1000
seq[i] ∈ [0,1]

OUTPUT:
Boolean value.
"True" if it is a prefix of Thue-Morse sequence. "false" otherwise.

Parameters
seq- This parameter is a integer array which contains the elements of a sequence with 0s and 1s (binary code).
Returns
Returns a boolean value (true or false) depending whether the binary code it a Thue-Morse.
